## 服务启动说明

---

## 每次启动项目

进入项目目录后，执行一条命令即可：

```bash
./启动所有服务.sh
```

就这么简单！脚本会自动启动所有服务（后端、前端、ngrok）。

该脚本会自动启动：
- 后端服务（端口 8000）
- 前端服务（端口 3000）
- ngrok 公网隧道（暴露前端）

---

## 访问地址

### 本地访问

- **前端页面**：http://localhost:3000
- **后端 API**：http://localhost:8000
- **API 文档**：http://localhost:8000/docs

### 公网访问（ngrok）

启动后会自动显示 ngrok 公网地址，或访问：
- **ngrok 管理界面**：http://127.0.0.1:4040

---

## 停止服务

```bash
pkill -f "uvicorn app.main:app"
pkill -f "vite.*--port 3000"
pkill -f "ngrok http"
```

---

## 查看日志

```bash
# 后端日志
tail -f logs/backend.out

# 前端日志
tail -f logs/frontend.out

# ngrok 日志
tail -f logs/ngrok.out
```
